# Markdown Learning Path

This guide lists the most recommended Markdown resources in learning order.

## 1) Beginner Start (Interactive Practice)

### Markdown Tutorial
[Markdown Tutorial](https://www.markdowntutorial.com/)

Each lesson focuses on one Markdown concept, then gives you a matching exercise right away. You can learn the core syntax in about 10 minutes, directly in the browser.

Additional notes:
- Best first stop if you are completely new to Markdown.
- No installation is required.

### CommonMark Interactive Tutorial
[CommonMark Tutorial](https://commonmark.org/help/tutorial/)

Another interactive course with concise examples in every lesson. You can jump between lessons and inspect example details quickly.

Additional notes:
- CommonMark is the standardized Markdown spec.
- Great for learning syntax that is portable across tools.

## 2) Systematic Reference (Save for Long-Term Use)

### Markdown Guide
[Markdown Guide](https://www.markdownguide.org/)

A free, open-source reference that covers beginner to advanced usage. The structure is clear:
- Getting Started
- Basic Syntax
- Extended Syntax (tables, fenced code blocks, footnotes, and more)

Additional notes:
- Use this as your long-term "dictionary".
- Ideal when writing README files and project documentation.

## 3) Quick Supplement

### W3Schools Markdown
[W3Schools Markdown](https://www.w3schools.com/markdown/)

A fast way to review syntax and examples when you need a quick reminder.

Additional notes:
- Useful for quick checks during real project work.
- Good as a backup reference when you want short examples.
